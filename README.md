## 正则表达式

### 正则表示基本用法

#### 元字符-字符匹配符
| 符号  | 含义        | 示例     | 解释            |
|-----|-----------|--------|---------------|
| []  | 可接收的字符列表  | [abc]  | a,b,c中的任意一个字符 |
| [^] | 不可接收的字符列表 | [^abc] | 不可接收字符：a,b,c  |
| -   | 连字符       | [a-z]  | a到z之间的任意小写字符  |

| 符号 | 含义 |
|-----|------|
|\\d|匹配任意一个数字字符|
|\\D|匹配任意一个非数字字符|
|\\w|匹配大小写字母，数字，下划线中的任意一个|
|\\W|匹配除大小写字母，数字，下划线之外的任意字符|
|\\s|匹配任意空格|
|\\S|匹配非空格符|
